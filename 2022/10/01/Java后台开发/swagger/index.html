Swagger

swagger 是在开发接口时一个接口集成管理工具,且承担测试接口的功能.

swagger 如何使用?

首先我们需要一个 spring boot 的项目,然后引入所需要的 swagger 依赖

当引入依赖后就可以使用了.
我们可以运行项目打开 swagger 接口集成页面地址: http://localhost/swagger-ui.html

注意:

如果不是多个 model 那么使用默认的请求地址即可
如果当前项目是 idea 中的model 那么需要将 model 的名称也加上
例如当前 model 名为 test那么请求地址就是 http://localhost/test/swagger-ui.html

接下来就是配置 swagger 的配置类,也就是在 spring boot 项目运行时会生成 swagger 的 Bean

swagger 有哪些常用注解?

@Api()

@ApiOperation()

@ApiModel()

@ApiModelProperty

@ApiImplicitParams 和 @ApiImplicitParam
